# 网页朗读助手（Chrome 扩展）

一个简单的谷歌浏览器插件：可朗读当前网页中的文字，并支持选择不同声音、语速、音调和音量。

## 功能

- 朗读当前网页主要内容（优先提取 `article`/`main`，否则使用 `body`）。
- 支持选择浏览器可用声音（不同系统会显示不同语音包）。
- 新增“中文增强音色”：自动为中文语音扩展温柔女声、清甜女声、主播音、沉稳男声等风格。
- 支持调节：
  - 语速（0.5 - 2.0）
  - 音调（0.0 - 2.0）
  - 音量（0.0 - 1.0）
- 支持开始、暂停/继续、停止。
- 自动记住上次设置。

## 安装方式（开发者模式）

1. 打开 Chrome，进入 `chrome://extensions/`。
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目目录。

## 使用

1. 打开任意网页。
2. 点击工具栏中的“网页朗读助手”图标。
3. 选择声音并调整参数。
4. 点击“开始朗读”。

## 项目结构

- `manifest.json`：扩展配置（Manifest V3）。
- `popup.html` / `popup.css` / `popup.js`：弹窗 UI 与交互逻辑。
- `content.js`：网页文本提取与语音朗读逻辑。

## 说明

- 插件使用浏览器内置 Web Speech API（`speechSynthesis`）。
- 可用声音由操作系统和浏览器决定；如果声音列表为空，可稍等几秒再打开弹窗重试。
基于Chrome浏览器，可语音朗读网页中的文字，并支持中文增强音色与多种音色切换。
